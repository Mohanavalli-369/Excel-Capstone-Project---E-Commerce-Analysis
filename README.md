E-Commerce Sales Analysis Dashboard
Project Overview

This project focuses on analyzing e-commerce sales data using Microsoft Excel. The objective of this project is to clean, transform, analyze, and visualize sales data to generate meaningful business insights and support data-driven decision-making.

The project includes:

Data Preparation
Data Cleaning
Data Modeling
Dashboard Creation
Sales Performance Analysis
Dataset Information

The dataset contains the following tables:

Customer_Dim
Product_Dim
Store_Dim
Sales_Fact

These tables were connected using relationships to create a star schema data model for effective analysis.

Data Cleaning and Preparation

The following preprocessing steps were performed:

Removed duplicate records
Handled missing values using mean and median methods
Standardized text values using TRIM and PROPER functions
Corrected inconsistent Product_ID formats
Validated row and column counts
Created derived columns such as Loyalty_Level using IF conditions
Converted raw data into structured Excel tables

Data Modeling

Relationships were created between dimension tables and the Sales_Fact table using primary and foreign keys.

Active relationships were maintained to ensure accurate reporting and filtering across the dashboard.
